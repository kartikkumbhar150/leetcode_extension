def twoSum(nums, target):
    seen = {}
    for i, n in enumerate(nums):
        diff = target - n
        if diff in seen:
            return [seen[diff], i]
        seen[n] = i

🔍 How It Works

From solve to GitHub commit
in seconds

01

You solve a problem on LeetCode

LeetSync's content script silently monitors your LeetCode tab for an accepted submission event.

→
02

Code is fetched via GraphQL

The service worker calls LeetCode's API to retrieve your submitted code, difficulty, and topic tags.

→
03

Pushed to your GitHub repo

Your code is committed under LeetCode/[Topic]/[problem]/Solution.[lang] using the GitHub REST API.

→
04

Revision is scheduled

LeetSync schedules 7 smart revision reminders so you never forget what you learned.

🚀 Installation

Get started in 3 simple steps

No Chrome Web Store required. Load it directly — takes under 2 minutes.

1

Download the Extension Files

Get the latest build from Google Drive — a pre-built dist/ folder ready to load.

Open Google Drive →
Download the entire folder as a ZIP, then extract it
a Click the link above to open Google Drive
b Right-click the folder → "Download" to download as ZIP
c Extract the ZIP to a permanent folder on your computer (e.g., Documents/LeetSync/)
2

Load Extension into Chrome

Enable Developer Mode and load the unpacked extension folder.

a Open Chrome and navigate to chrome://extensions
b Toggle Developer mode ON (top-right corner)
c Click "Load unpacked" button (top-left)
d Navigate to and select the extracted folder (the one containing manifest.json)
e The LeetSync ⚡ icon will appear in your Chrome toolbar

3

Connect Your GitHub Account

Generate a Personal Access Token and link it to LeetSync — one-time setup.

a Go to github.com/settings/tokens
b Click "Generate new token (classic)"
c Give it a name (e.g., leetsync) and check the ✅ repo scope
d Click "Generate token" and copy the token (starts with ghp_...)
e Click the ⚡ LeetSync icon in Chrome toolbar → go to Settings
f Enter your GitHub Token, Username, and the Repository name you want to use
g Click "Save Settings" — you're all set! 🎉
Important: Create the GitHub repository first (can be empty). LeetSync will populate it automatically after your first accepted submission.
📖 Usage Guide

How to use every feature

🔄

Auto GitHub Commit

Once configured, LeetSync works automatically. Just solve problems on LeetCode as usual — whenever a submission is accepted, LeetSync commits it to your repo within seconds. No button to click, no copy-paste.

✅ Works on all LeetCode problem pages automatically
📊

Dashboard & Analytics

Click the ⚡ LeetSync icon in your toolbar → click "Open Dashboard". You'll see:

  • Difficulty breakdown (Easy / Medium / Hard)
  • Topic radar chart (DP, Graphs, Trees, etc.)
  • Calendar heatmap of your activity
  • Current & longest streak counters
  • Average solve time per difficulty
🧠

Revision System

Navigate to the Revisions tab in the dashboard. Problems due for revision appear at the top.

  • Click ✅ Remembered to advance to the next revision date
  • Click ❌ Forgot to reset that slot to tomorrow
  • LeetSync will notify you each day when revisions are due

Coding Journal

Go to the Journal tab to see every problem you've solved, along with:

  • Date and time of submission
  • Problem difficulty and topic tags
  • Time spent on the problem
  • Your personal notes (mistakes, patterns, observations)

Click any entry to expand and add or edit your notes.

⚙️

Settings & Configuration

Click ⚡ LeetSync → Settings to manage:

  • GitHub Token — your personal access token
  • GitHub Username — your GitHub handle
  • Repository Name — the repo to push solutions to
  • Notifications — toggle daily revision reminders
⚠️ Never share your GitHub token. It grants repo access.
